How to Connect

You can connect to our server using any software that works with Ergo. T-Rex Miner will allow you to overclock within the software, or you can use NBminer with MSI afterburner for NVidia cards. AMD works great with TeamRedMiner

Server: stratum+tcp://tulipfarm.one:8008

No password is used, just put your wallet address and an optional worker name. Follow the examples in the software you choose.
